## How to Add Screenshots

1. Take screenshots of the features in action
2. Save them with the exact filenames listed above
3. Place them in this directory
4. Ensure images are high quality (PNG format preferred)
5. Recommended resolution: 1920x1080 or higher

## Screenshot Guidelines

- Use high-quality, clear screenshots
- Show the feature in action, not just static UI
- Include relevant context (file names, code snippets, etc.)
- Ensure no sensitive information is visible
- Use consistent window sizes and themes
