## ‚ö° Quick Demo (5 Minutes)

### 1. Visit the Live Demo

üîó [https://aiwa-ai-demo.vercel.app](https://aiwa-ai-demo.vercel.app)

### 2. Try the Prompt Analyzer

1. Click "New Chat"
2. Type: "app" in the prompt box
3. Watch the real-time analysis (red scores = needs improvement)
4. Now type: "Create a React TypeScript todo app with authentication"
5. Watch scores turn green!

### 3. Create a Project

1. Click "Projects" in the sidebar
2. Click "New Project"
3. Add a project name
4. Add an environment variable (e.g., API_KEY=test123)
5. Create a chat under this project
6. The AI will automatically use your environment variables!

### 4. Try Multi-Device Preview

1. Generate any app
2. Click the device icons (mobile/tablet/desktop)
3. See instant responsive preview
4. Check the console tab for live logs

### 5. Export Your Code

1. Click "Export" button
2. Choose "Download ZIP" or "Export to GitHub"
3. Notice the .env file is included!
